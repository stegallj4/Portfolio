Admin Site with Dark/Light Mode

2024

JavaScript

This site was the first of my personal projects that I put together. Utilizing CSS Flexbox and Media Queries, this admin site is fully responsive for any window shape or size on a desktop. With a combination of over 300 lines of HTML, 700 lines of CSS, and only 33 lines of JavaScript, DOM Manipulation and Event Listeners were employed to create a seamless transition from Light to Dark Mode.
**This project was not included in the Nucamp Coding Bootcamp curriculum.
